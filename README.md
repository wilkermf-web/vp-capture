# vp-capture

Captura **página completa** do Vaidepromo (autorizado) para datas de voo e salva no repositório.

## Como rodar

1. Vá em **Actions → Capture Vaidepromo → Run workflow**.
2. Preencha:
   - `origin` (ex.: GYN)
   - `dest`   (ex.: CAC)
   - `date`   (YYYY-MM-DD)
